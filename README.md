# 関西Ruby会議09 / KansaiRubyKaigi09

https://regional.rubykaigi.org/kansai09/

## Setup

```bash
npm install
```

## Development Server

```bash
npm run dev
```

Navigate to `http://localhost:3000`

## Production

Build the application for production:

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```

## Generate Static Site

```bash
npm run generate
```
